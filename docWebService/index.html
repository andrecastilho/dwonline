Manual de acesso ao Web Service DwOnline

Acesso ao WebService

O webservice DwOnline pode ser acessado atravez da url: http://dwonline.com.br/desenv/serviceDataWeb/webService/, onde terá acesso as funções abaixo:

consultaCnpj
consultaCpf
consultaTelefone
consultaNome
consultaEndereco
consultaCnf

Será necessário algumas informações e créditos disponíveis para o retorno das buscas em nosso serviço.

Estas informações devem ser passadas via GET ou POST:

consultaCnpj : parametros: tns:arrConsultaCnpj

array( "login" => ['login'], "cnpjEmpresa" => ['cnpjEmpresa'], "cnpj" => ['cnpj']);

Onde :
   longin = Neste parametro deverá ser informado o código de liberação do serviço, este código será enviado via e-mail ao responsavel pela conta.
   cnpjEmpresa = Cnpj informado no cadastro da conta.
   cnpj = Deverá ser informado o CNPJ no qual se busca informações.

consultaCpf : parametros: tns:arrConsultaCpf

array( "login" => ['login'], "cnpjEmpresa" => ['cnpjEmpresa'], "cpf" => ['cpf']);

Onde :
   longin = Neste parametro deverá ser informado o código de liberação do serviço, este código será enviado via e-mail ao responsavel pela conta.
   cnpjEmpresa = Cnpj informado no cadastro da conta.
   cpf = Deverá ser informado o CPF no qual se busca informações.

consultaTelefone : parametros: tns:arrConsultaTelefone

array( "login" => ['login'], "cnpjEmpresa" => ['cnpjEmpresa'], "telefone" => ['telefone'], "ddd"=>['ddd']);

Onde :
   longin = Neste parametro deverá ser informado o código de liberação do serviço, este código será enviado via e-mail ao responsavel pela conta.
   cnpjEmpresa = Cnpj informado no cadastro da conta.
   telefone = Deverá ser informado o telefone(apenas numero) no qual se busca informações.
   ddd = Deverá ser informado o dd (apenas numeros) no qual se busca informações.

consultaNome : parametros: tns:arrConsultaNome

array( "login" => ['login'], "cnpjEmpresa" => ['cnpjEmpresa'], "nome" => ['nome']);

Onde :
   longin = Neste parametro deverá ser informado o código de liberação do serviço, este código será enviado via e-mail ao responsavel pela conta.
   cnpjEmpresa = Cnpj informado no cadastro da conta.
   nome = Deverá ser informado o nome da pessoa física no qual se busca informações.

consultaEndereco : parametros: tns:arrConsultaEndereco

array( "login" => ['login'], "cnpjEmpresa" => ['cnpjEmpresa'], "endereco" => ['endereco'] "numero" => ['numero'] "bairro" => ['bairro'] "cidade" => ['cidade'] "uf" => ['uf'] "cep" => ['cep'] );

Onde :
   longin = Neste parametro deverá ser informado o código de liberação do serviço, este código será enviado via e-mail ao responsavel pela conta.
   cnpjEmpresa = Cnpj informado no cadastro da conta.
   endereco = Deverá ser informado o endereco da pessoa física no qual se busca informações.
   numero = Deverá ser informado o numero residencial da pessoa física no qual se busca informações.
   bairro = Deverá ser informado o bairro da pessoa física no qual se busca informações.
   cidade = Deverá ser informado a cidade da pessoa física no qual se busca informações.
   uf = Deverá ser informado o estado (ex: sp,mg,ba) da pessoa física no qual se busca informações.
   cep = Deverá ser informado o cep residencial da pessoa física no qual se busca informações.

consultaNome : parametros: tns:arrConsultaNome

array( "login" => ['login'], "cnpjEmpresa" => ['cnpjEmpresa'], "nome" => ['nome']);

Onde :
   longin = Neste parametro deverá ser informado o código de liberação do serviço, este código será enviado via e-mail ao responsavel pela conta.
   cnpjEmpresa = Cnpj informado no cadastro da conta.
   nome = Deverá ser informado o nome da pessoa física no qual se busca informações.

consultaCnf parametros: tns:arrConsultaCnf

array( "login" => ['login'], "cnpjEmpresa" => ['cnpjEmpresa'], "cpf" => ['cpf'] );

Onde :
   longin = Neste parametro deverá ser informado o código de liberação do serviço, este código será enviado via e-mail ao responsavel pela conta.
   cnpjEmpresa = Cnpj informado no cadastro da conta.
   cpf = Deverá ser informado o cpf da pessoa física no qual se busca informações no banco de dados do CNF.


Exemplo: dwonline/serviceDataWeb/consultaCpf/exemploPhp/

Retorno webservice DwOnline

   Retornos de erros

Serão retornados códigos caso algo impeça o envio das informações. Estes códigos são:

001 - Falha na autenticação
002 - Nada há retornar.
003 - Sem créditos para operação
